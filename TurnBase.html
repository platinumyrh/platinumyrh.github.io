秘箭连击：移动端 Boss Rush

(Arrow Rush: Mobile Boss Battle) - 动效优化与数据驱动的数值系统

本项目是一款移动端回合制 Boss Rush 游戏，重点实现核心战斗连击机制、流畅的受击动效和可扩展的局外升级系统。

核心功能与系统实现

  • **核心战斗机制：** 实现了基于随机箭头输入的回合制连击系统，玩家需在限定时间内准确输入指令以积累连击数，对 Boss 造成爆发伤害。
  • **视觉反馈优化 (DOTween)：** 广泛使用 **DOTween** 插件实现敌我双方的受击反馈、UI 弹跳、连击数字跳动等动效，极大地提升了战斗表现力和打击感。
  • **统一数值管理：** 设计并实现了 **`PlayerStats`** 类，用于集中处理玩家的攻击力、防御力、生命值等所有属性计算和升级逻辑，确保数值系统的统一性。
  • **局外成长系统：** 实现了基于 Boss 掉落金币的属性升级系统，通过本地持久化存储玩家的成长数据，提供长期驱动力。

技术栈与工具

Unity 2022+ C# DoTween Plugin UI/UX Design (Mobile) OOP Data Structure Git 版本控制
