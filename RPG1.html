遗迹探险家：核心框架

(RPG Core Framework) - 模块化与数据驱动的系统设计

本项目旨在构建一套可复用、易扩展的 Unity RPG 核心系统，重点关注 AI 逻辑、数据管理和本地持久化。

核心功能与系统实现

  • **模块化 AI 系统 (C# Behavior Tree)：** 实现了自定义的**行为树**结构，用于管理敌人 NPC 的巡逻（Patrol）、追逐（Chase）和攻击（Attack）状态。通过面向对象设计，使得行为节点可插拔、易于调试。
  • **数据驱动设计 (Scriptable Objects)：** 所有非代码数据，如对话文本、任务定义、物品属性和角色基础数值，均使用 **Unity Scriptable Objects** 进行管理。极大地提高了策划配置效率和数据的可维护性。
  • **轻量级本地存档：** 利用 C# 内置的 `JsonUtility` 库，实现了玩家背包、任务进度和属性数据的 To-JSON 序列化与反序列化，实现可靠、轻量的本地存档功能。
  • **事件系统 (C# Events)：** 使用 C# 事件和委托（Delegates）解耦 UI 逻辑和核心系统逻辑（例如：任务更新、物品拾取），降低了模块间的依赖性。

技术栈与工具

Unity 2022+ C# Scriptable Objects Behavior Tree JSON Utility C# Events/Delegates 2D/3D 移动控制 面向对象设计 (OOP)
