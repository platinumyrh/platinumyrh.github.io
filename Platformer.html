像素探险：2D平台跳跃游戏

(Pixel Explorer: 2D Platformer) - 物理驱动的流畅操控与关卡设计

本项目旨在开发一款 2D 像素风格的平台跳跃游戏，重点实现高响应度的角色控制器、复杂的移动机制和模块化关卡组件。

核心功能与系统实现

  • **高响应度角色控制器：** 自定义实现了基于物理学的角色移动逻辑，包括跳跃、二段跳、冲刺和贴墙滑行，确保了流畅和精确的操作手感。
  • **精确碰撞检测 (Raycasting)：** 采用射线投射（Raycasting）机制处理地面、墙壁和障碍物的精确碰撞，解决了 Unity 默认物理引擎在快速移动中可能出现的穿透问题。
  • **状态机 (FSM) 管理：** 使用简单的状态机模式管理角色状态（Idle, Run, Jump, Dash, Hit），确保状态切换逻辑清晰、可控，动画播放与状态同步。
  • **模块化关卡组件：** 设计了可复用的关卡元素（移动平台、陷阱、收集品），通过 Unity Prefabs 和接口实现交互逻辑，极大提高了关卡设计效率。
  • **时间回溯机制 (Time Rewind):** 通过周期性地存储角色的位置、旋转、Animator 状态等关键帧数据，并在回溯时反向读取并应用这些数据，实现了精确且流畅的“时间回溯”效果。

技术栈与工具

Unity 2022+ C# 2D Physics/Raycasting State Machine (FSM) Unity Tilemap 面向对象设计 (OOP) Git 版本控制

项目贡献

  • **角色控制器**：独立编写 (100%)
  • **移动机制**：二段跳、冲刺、贴墙滑行逻辑实现 (100%)
  • **状态机实现**：管理动画与逻辑同步 (100%)
  • **关卡组件**：移动平台、陷阱交互逻辑 (100%)
